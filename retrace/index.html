Screen-Recall is a small local app: it captures the screen every few seconds, reads the text with OCR, and makes every moment searchable by keyword and by meaning. This page runs the exact same lexical, semantic, and fusion math, ported to JavaScript, against one invented workday — the algorithms are real, nothing on this page was ever actually captured.

On load, this page replayed all six search chips through the in-browser engine (retrace.js) and compared ranked-id order and fused RRF scores against a reference computed by the real Python search code (build_corpus.py — real BM25 via SQLite FTS5, real MiniLM embeddings, real rrf_fuse). The same check runs offline via node demo/selftest.node.js in the repo.
